Question-1: What are the different ways to select an element in the DOM?

There are 5 ways that we can use such as:
1. document.getElementById('id'); This method selects a single element by its unique ID attribute.
2. document.getElementsByClassName('className'); This method selects elements based on class name attribute.
3. document.getElementsByTagName('tagName'); This method selects elements based on tag name.
4. document.querySelector('Selector'); This method select the first element that match specific selector.
5. document.querySelectorAll('Selector'); It returns a Node list of all elements that match the specified css selector.

Question-2: What is the difference between innerHTML, innerText and textContent?

Firstly here innerHTML returns the full HTML content, including tags. Secondly innerText returns only the visible, rendered text, respecting CSS but ignoring hidden content and tags. At last textContent returns all text, including hidden text and spacing

Question-3: What is event delegation in the DOM?

Event Delegation is principally a pattern to handle events efficiently. rather of adding an event listener to each and every analogous element, we can add an event listener to a parent element and call an event on a particular target using the. target property of the event object.

Question-4: What is event bubbling in the DOM?

Event bubbling is a fundamental JavaScript mechanism where an event triggered on a child element does not stop there, but "bubbles up" through its ancestor elements in the Document Object Model (DOM).
